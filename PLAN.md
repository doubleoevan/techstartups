## Session Start — Always Do This First

1. Read Notion > Sprint/Tasks for current status and active tasks
2. Read Notion > Architecture for monorepo/stack context
3. Read Notion > Decision Log for relevant decisions
4. Ask the user what today's goal is if not already stated

## Session End — Always Do This Last

1. Update Notion > Sprint/Tasks with what was completed and what's next
2. Log any new decisions made to Notion > Decision Log
